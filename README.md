# Tetris

Запуск:\
1) npm i;\
2) npm start.

Управление:
&#8592; - влево;
&#8594; - вправо;
&#8593; - поворот по часовой стрелке;
&#8595; - поворот против часовой стрелки;
space - опустить блок;
ctrl+c - закрыть.

Правила:
поле 15x10;
за каждую полностью заполненную строчку начисляется 1 балл;
игра заканчивается, если новый блок сразу упирается в предыдущие.
